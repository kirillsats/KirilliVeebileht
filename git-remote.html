Kuidas teha remote repo GitHubis?

Kui soovid luua ja ühendada kaugrepo (remote repository) GitHubis, järgi neid samme:

1. Loo uus repository GitHubis

Mine GitHubi ja loo uus repository. Ära lisa README, .gitignore või License faili.

2. Loo ja ühenda lokaalne repository

git init
git add .
git commit -m "Esimene commit"

3. Lisa remote repository ja pushi andmed

git remote add origin https://github.com/KASUTAJA/NIMI.git
git branch -M main
git push -u origin main

Asenda KASUTAJA oma GitHubi kasutajanimega ja NIMI repository nimega.
